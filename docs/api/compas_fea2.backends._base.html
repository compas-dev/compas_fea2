Base

This is the Base implementation of compas_fea2, where the individual backends inherit from.

Elements

A 1D element that resists axial, shear, bending and torsion.

TrussElementBase(connectivity, section[, …])

A 1D element that resists axial loads.

ShellElementBase(connectivity, section[, …])

A 2D element that resists axial, shear, bending and torsion.

MembraneElementBase(key, connectivity, section)

A shell element that resists only axial loads.

SolidElementBase(key, connectivity, section)

A 3D element that resists axial, shear, bending and torsion.
